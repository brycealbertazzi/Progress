# Progress

A workout logging app for iOS and Android built with Flutter. Track your exercises, log sets/reps/weight, and visualize your progress over time.

## Features

- Create and manage custom exercises
- Log sets, reps, and weight for each session
- View progress history per exercise over time

## Tech Stack

- **Framework**: Flutter
- **Platforms**: iOS & Android
- **Bundle ID**: `com.albertazzilabs.progress`

## Getting Started

### Prerequisites

- [Flutter SDK](https://docs.flutter.dev/get-started/install) (3.x or later)
- Xcode (for iOS)
- Android Studio (for Android)

### Running the app

```bash
flutter pub get
flutter run
```

To run on a specific platform:

```bash
flutter run -d ios
flutter run -d android
```
